================================================================================
COMPLETE EXERCISE SYSTEM - GUIDE
================================================================================

SYSTEM OVERVIEW:
================

This system provides real-time exercise validation with voice feedback for
physical therapy. It uses YOLOv8 Pose detection + joint angle analysis +
error detection + voice recommendations.

================================================================================
FILES (5 TOTAL):
================================================================================

1. voice_feedback.py         - Voice engine with cooldown
2. exercise_rules.py          - Exercise definitions and error rules
3. exercise_correction.py     - Main correction logic
4. complete_system.py         - Complete working example
5. SYSTEM_GUIDE.txt          - This documentation

================================================================================
QUICK START:
================================================================================

STEP 1: Install
    pip install pyttsx3

STEP 2: Run
    python complete_system.py

STEP 3: Select exercise (1-8) and start!

Controls:
    Q = Quit
    R = Reset counters
    V = Toggle voice on/off

FILE 1: voice_feedback.py
================================================================================

CLASS: VoiceFeedback

    __init__(rate, volume, cooldown)
        rate: Speech speed (default 150)
        volume: Volume 0.0-1.0 (default 0.9)
        cooldown: Seconds between same message (default 3.0)

    speak(text, force) -> bool
        text: Message to speak
        force: Skip cooldown if True
        Returns True if spoken, False if blocked

    enable()
        Turn voice on

    disable()
        Turn voice off

    clear_cooldown()
        Reset cooldown cache

    stop()
        Stop TTS engine

USAGE:
    from voice_feedback import VoiceFeedback

    voice = VoiceFeedback(cooldown=3.0)
    voice.speak("Bend your knee more")
    voice.disable()
    voice.enable()

FILE 2: exercise_rules.py
================================================================================

CLASS: ErrorDetection (dataclass)
    error_type: str
    severity: str ('low', 'medium', 'high')
    recommendation: str
    joint_name: str
    current_value: float
    expected_range: tuple

DICTIONARY: EXERCISE_RULES

Contains rules for 8 exercises:

1. bending_knee_no_support_seated
   Joint: knee (70° - 180°)
   Errors: insufficient_flexion, excessive_flexion, insufficient_extension, excessive_extension
   Speed check: Yes (min 2.0 sec)

2. bending_knee_bed_support_supine
   Joint: knee (60° - 150°)
   Errors: insufficient_flexion, excessive_flexion, insufficient_extension
   Speed check: Yes (min 2.5 sec)

3. bending_knee_with_support_seated
   Joint: knee (80° - 160°)
   Errors: insufficient_flexion, insufficient_extension
   Speed check: Yes (min 2.0 sec)

4. circular_pendulum_standing
   Joint: shoulder (10° - 60°)
   Errors: excessive_motion, insufficient_motion, stopped_motion
   Speed check: No (continuous motion)

5. external_rotation_shoulders_elastic
   Joint: elbow (70° - 110°)
   Errors: elbow_too_open, elbow_too_closed, fast_motion
   Speed check: Yes (min 2.5 sec)

6. horizontal_weighted_openings_standing
   Joint: shoulder (60° - 120°)
   Errors: insufficient_opening, excessive_opening, insufficient_closing
   Speed check: Yes (min 2.0 sec)

7. lift_extended_leg_supine
   Primary: hip (30° - 70°)
   Secondary: knee (160° - 180°, must stay straight)
   Errors: leg_not_straight, insufficient_lift, excessive_lift, leg_lowered_incomplete
   Speed check: Yes (min 2.5 sec)

8. shoulder_flexion_seated
   Joint: shoulder (10° - 170°)
   Errors: insufficient_raise, excessive_raise, incomplete_lowering
   Speed check: Yes (min 2.0 sec)

STRUCTURE:
    {
        'joint': 'knee',
        'min_angle': 70,
        'max_angle': 180,
        'errors': {
            'error_name': {
                'condition': lambda angle, min_a, max_a: angle > min_a + 20,
                'recommendation': "Bend your knee more",
                'severity': 'medium'
            }
        },
        'movement_speed_check': True,
        'min_rep_duration': 2.0
    }

USAGE:
    from exercise_rules import EXERCISE_RULES, ErrorDetection

    rules = EXERCISE_RULES['shoulder_flexion_seated']
    print(rules['min_angle'])
    print(rules['max_angle'])

FILE 3: exercise_correction.py
================================================================================

CLASS: ExerciseCorrection

    __init__(voice_enabled, cooldown)
        voice_enabled: Enable voice (default True)
        cooldown: Seconds between messages (default 3.0)

    detect_errors(exercise_name, joint_angles, current_rep_state, is_valid_motion, side)
        exercise_name: Exercise key from EXERCISE_RULES
        joint_angles: {'left_knee': 90.0, 'right_knee': 180.0}
        current_rep_state: 'min', 'max', 'transition', 'neutral'
        is_valid_motion: True/False
        side: 'left' or 'right'
        Returns: List of ErrorDetection objects

    generate_recommendation(errors)
        errors: List of ErrorDetection
        Returns: Highest priority recommendation string

    provide_feedback(exercise_name, joint_angles, current_rep_state, is_valid_motion, side, force_speak)
        Returns: {
            'errors': [ErrorDetection, ...],
            'recommendation': "Bend your knee more",
            'spoken': True/False,
            'error_count': 2,
            'highest_severity': 'medium'
        }

    enable_voice()
        Turn voice on

    disable_voice()
        Turn voice off

    reset()
        Clear all tracking

    get_statistics()
        Returns error statistics

FUNCTION: create_correction(voice_enabled, cooldown)
    Factory function to create ExerciseCorrection instance

USAGE:
    from exercise_correction import create_correction

    correction = create_correction(voice_enabled=True, cooldown=3.0)

    feedback = correction.provide_feedback(
        exercise_name='shoulder_flexion_seated',
        joint_angles={'left_shoulder': 95.0},
        current_rep_state='min',
        is_valid_motion=False,
        side='left'
    )

    print(feedback['recommendation'])
    print(feedback['spoken'])

EXAMPLE 2: With Pose Validation
--------------------------------

from ultralytics import YOLO
from exercise_validator import ExerciseValidator
from exercise_correction import create_correction

pose_model = YOLO('yolov8n-pose.pt')
validator = ExerciseValidator('shoulder_flexion_seated')
correction = create_correction()

cap = cv2.VideoCapture(0)

while True:
    ret, frame = cap.read()

    pose_results = pose_model(frame)
    keypoints = pose_results[0].keypoints.data[0].cpu().numpy()

    validation = validator.validate_frame(keypoints, 'left')

    joint_angles = {'left_shoulder': validation.current_angle}

    state_map = {
        'IDLE': 'neutral',
        'MIN_POSITION': 'min',
        'MAX_POSITION': 'max',
        'TRANSITION': 'transition'
    }

    rep_state = state_map.get(validation.state.value, 'neutral')

    feedback = correction.provide_feedback(
        exercise_name='shoulder_flexion_seated',
        joint_angles=joint_angles,
        current_rep_state=rep_state,
        is_valid_motion=validation.is_valid,
        side='left'
    )

================================================================================
PARAMETERS REFERENCE:
================================================================================

joint_angles format:
    {'left_knee': 90.0, 'right_knee': 180.0}
    {'left_shoulder': 45.0}
    {'left_hip': 60.0, 'left_knee': 175.0}

current_rep_state values:
    'min' = At minimum angle (bent)
    'max' = At maximum angle (extended)
    'transition' = Moving between
    'neutral' = Idle/invalid

side values:
    'left' or 'right'

severity values:
    'low' = Minor issue
    'medium' = Important correction
    'high' = Critical safety issue

================================================================================
TROUBLESHOOTING:
================================================================================

PROBLEM: Voice not speaking
SOLUTION:
    - pip install pyttsx3
    - Linux: sudo apt-get install espeak
    - Check audio not muted
    - Test: python -c "import pyttsx3; pyttsx3.init().say('test')"

PROBLEM: Too many messages
SOLUTION:
    - Increase cooldown: create_correction(cooldown=5.0)
    - Adjust error thresholds in exercise_rules.py

PROBLEM: Import errors
SOLUTION:
    - Ensure all files in same directory
    - Check exercise_validator.py exists
    - Verify: python -c "import exercise_correction"

PROBLEM: Wrong recommendations
SOLUTION:
    - Check joint_angles format: {'left_knee': 90.0}
    - Verify exercise_name matches EXERCISE_RULES keys
    - Ensure angles in degrees (0-180)

PROBLEM: No errors detected
SOLUTION:
    - Verify joint_angles has correct side prefix
    - Check exercise_name spelling
    - Debug: print(joint_angles)

PROBLEM: Webcam not working
SOLUTION:
    - Try: cv2.VideoCapture(1)
    - Check permissions
    - Test: python -c "import cv2; print(cv2.VideoCapture(0).isOpened())"

================================================================================
WORKFLOW:
================================================================================

1. System loads YOLOv8 Pose + Validator + Correction
2. Camera captures frame
3. YOLOv8 detects person and keypoints
4. Validator calculates joint angles
5. Validator checks angle ranges
6. Correction detects errors
7. Correction generates recommendation
8. Voice speaks recommendation (with cooldown)
9. Display shows angles, reps, feedback
10. Repeat
